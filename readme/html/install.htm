环境要求

操作系统要求

    Windows Server 2008(推荐)、Windows Server 2003、Windows XP

语言及数据库支撑环境要求

    安装Discuz!NT 需要装有如下软件：

    • IIS 5.0及以上
    • .NET Framework 2.0及以上
    • SQL Server 2000及以上

    如果您租用虚拟主机，请咨询虚拟主机提供商，您的空间服务器是否已安装了上述软件。由于 Discuz!NT 的数据表具有前缀设计，因 此通常情况下可以将 Discuz!NT 与其他软件安装在同一个数据库中，或采用不同的前缀名从而在同一个数据库中安装多个 Discuz!NT 论坛而 不产生冲突。

    您的 SQL Server数据库账号应当拥有"当前数据库管理员"权限，同时文件空间不低于 5M，数据库空间不低于 10M，通常您的 虚拟空间都会满足这个条件，以满足包括 Discuz!NT 在内的绝大多数网络软件的正常运行。如果您不了解具体情况，请咨询您的空间提供 商。

安装可能用到的工具软件

    Discuz!NT 开发组尽量使得安装步骤简单方便，但仍然可能会用到一些常用的工具软件。如果您通过网络将 Discuz!NT 软件上传到服务器 上，您将可能需要一个 FTP 客户端软件。通过您的服务器 FTP 账号，使用该 FTP 客户端软件将相关文件上传到服务器上。同时您可能 需要一个简单的文本文件编辑软件，用以对配置文件进行参数修改，一般操作系统自带的如 Windows 的“记事本”或第三方软件如 UltraEdit 等都能胜任。
