无法使用Discuz!NT代码
为什么我无法使用Discuz!NT代码、[img]标签、表情等

      您在论坛使用中可能无法在帖子中插入图片，对文字进行加粗，变更字体等操作。

      遇到上述问题时您应该检查您论坛 版块设置 -> 论坛编辑 -> 高级设置 -> Discuz!NT代码是否选中？如果没有选中则您无权在版块中使用Discuz!NT代码。需要使用Discuz!NT代码则请您开启此项。（[img]标签，表情等请参考此设置。）


无法正常登录论坛
1.用户在发帖时提示权上传附件。

调整您后台分组与级别 -> 用户组编辑 -> 调整用户组的权限，选中是否上传附件项即可。

2.我上传附件，提示我附件尺寸超出限制。

 在Discuz!NT正式版时，统一了附件尺寸按照字节进行运算。
您出现此问题是可以检查您后台 帖子相关 -> 附件类型尺寸中附件的尺寸，修改为对应的大小后即可恢复正常。
以下是常用尺寸的换算结果：
1MB = 1024000字节 
2MB = 2048000字节 
4MB = 4096000字节 

3.上传附件时提示我http请求超时。
 
如果有http超时的提示请在web.config中的<system.web>节中添加<httpRuntime maxRequestLength="1000000" executionTimeout="1800" />来增长http链接在上传文件时的超时时间。
 
4.我每天允许上传附件的总量为0导致无法上传。 

调整您后台分组与级别 -> 用户组编辑 -> 调整用户组中“每天最大附件总尺寸:”的设置即可。此项不能为0，切记是按照字节进行统计的。


我想注册的用户名无法注册
在论坛后台的分组与组别 -> 系统组编辑 -> 编辑游客组，将阅读权限设置为-1即可。


我为会员提升了用户组，但他登录后又变成了原来的组
您首先需要在后台分组与级别选项中设置用户组使用Discuz!NT代码以及，hide标签的使用权限。
然后您就可以在发帖时使用[hide][/hide]来发表帖子。例如:测试隐藏[hide]隐藏内容[/hide]，此时用户将无法直接查看[hide][/hide]标签中的内容，必须在对帖子进行回复后才可以浏览。
